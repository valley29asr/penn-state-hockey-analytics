# 🏒 Penn State Hockey Analytics

An end-to-end hockey analytics project built using Python, machine learning, and interactive dashboards to analyze team performance and identify the biggest drivers of winning hockey games.

## Project Goals

- Build a clean game-level hockey dataset
- Engineer advanced hockey performance metrics
- Analyze what contributes most to winning
- Build predictive machine learning models
- Develop an interactive dashboard for coaches and analysts

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Plotly
- Streamlit
- Git & GitHub
